# Healthcare System — Backend

A microservices-based backend for managing patients and billing in a healthcare context. Built with Spring Boot, gRPC, Apache Kafka, and Docker.

> **Status:** In active development

---

## Tech Stack

- **Java 17 / 21** — Spring Boot 
- **Spring Data JPA** — persistence layer
- **PostgreSQL** — production database 
- **gRPC + Protocol Buffers** — inter-service communication
- **Apache Kafka** — event-driven patient lifecycle events
- **Docker** — containerized deployment

---

### Prerequisites
- Java 17+
- Maven 3.8+
- Docker & Docker Compose
- Kafka (local or via Docker)
